# Task 02 — Medical Appointment Booking System

## Project Overview

This project focuses on designing a mobile appointment booking experience for an eye surgeon.

The objective was to create a structured, calm, and trustworthy digital flow that reduces patient anxiety while maintaining clarity in medical scheduling.

Healthcare booking is not a simple transactional interaction — it is often the first point of trust between a patient and a specialist.

---

## Problem Statement

Existing medical booking systems often face:

- Unclear availability visibility
- Multi-step onboarding friction
- Weak confirmation reassurance
- Limited trust-building signals

For specialist care, these issues increase cognitive load and reduce booking confidence.

---

## Solution Approach

The solution prioritizes:

- Calendar-first slot visibility
- Minimal step booking flow
- Secure OTP-based verification
- Clear confirmation with downloadable appointment record

Each interaction was designed to reduce friction and maintain reassurance throughout the journey.

---

## Key Features

- Secure account creation & verification
- Structured calendar-based date selection
- Time-slot clarity
- Downloadable appointment confirmation
- Clean visual hierarchy

---

## UX Principles Applied

- Reduced cognitive load
- Progressive disclosure
- Clear visual hierarchy
- Trust reinforcement through structured feedback
- Minimal interface noise

---

## Tools Used

- Figma

---

## Outcome

The final system delivers a calm, structured appointment booking experience aligned with healthcare usability standards and trust-driven interaction design.
