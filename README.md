# FUTURE UI/UX Internship Submission

This repository contains the completed design tasks as part of the Future UI/UX Internship.

The focus of this internship was to solve real-world product problems using structured UI/UX thinking and system-driven design.

---

## Tasks Overview

### Task 01 — UI/UX Case Study
Focused on applying core UI/UX principles including hierarchy, layout structure, spacing consistency, and usability fundamentals.

### Task 02 — Medical Appointment Booking System
Designed a mobile booking experience for an eye surgeon, prioritizing clarity, reduced cognitive load, and trust-driven interaction.

### Task 03 — CRM System for a UI/UX Agency
Built a structured operational CRM system to manage leads, clients, projects, revenue tracking, and task execution.

---

## Tools Used

- Figma

---

## Design Approach

Across all tasks, the focus remained on:

- System thinking over isolated screens
- Clear information hierarchy
- Functional clarity over decoration
- Scalable structure and modular architecture
